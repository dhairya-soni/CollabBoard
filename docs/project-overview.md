# Project Overview

## What is CollabBoard?

CollabBoard is a **full-stack collaborative project management tool** featuring:
- Kanban board with drag-and-drop task management
- Infinite canvas whiteboard for visual brainstorming
- Real-time collaboration (live cursors, live updates, presence indicators)
- AI-powered task breakdown

## Purpose

This is an **SDE portfolio project** built to demonstrate production-level full-stack engineering skills. The quality bar is premium SaaS products like **Linear** and **Vercel** — not a typical tutorial project.

## Target Users

- Small to mid-size development teams
- Product managers tracking sprint work
- Designers collaborating with engineers via the whiteboard

## Key Differentiators

1. **Real-time everything** — See other users' cursors, live task movements, typing indicators
2. **Infinite canvas** — Draw, link sticky notes to tasks, brainstorm visually
3. **Premium feel** — Smooth animations, keyboard shortcuts, command palette
4. **Production-grade** — Auth, error tracking, CI/CD, monitoring, testing
